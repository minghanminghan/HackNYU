<h1>Overview</h1>
Data viewer controlled by hand movements

<h1>Usage</h1>
1. `git clone https://github.com/minghanminghan/hacknyu/tree/v2.1/`
2. `cd hacknyu`
3. `python cli.py read data.csv`
